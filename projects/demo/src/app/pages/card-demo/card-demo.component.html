Card

pupa-card is a flexible container component.

It can be used alone, but it also provides several common pre-styled card sections:

  • pupa-card-container - container for card content used inside pupa-card but can be used alone
  • pupa-card-cover - container for icon/image or any other content
  • pupa-card-header - container for card header content
  • pupa-card-title - container for card title
  • pupa-card-description - container for card description
  • pupa-card-action - container for action button

All these sections are optional.
