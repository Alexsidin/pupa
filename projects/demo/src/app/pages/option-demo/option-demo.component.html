Option

pupa-option is a configurable option component.

There are range of components that are used as content to build and configure the option itself:

  • pupa-option-title - a wrapper to truncate long text
  • pupa-option-description - a wrapper to truncate long description
  • pupa-option-icon - a wrapper to adding icon
This is a visual component. Activity state changes outside. You can see it on tab Api.

Directive *pupaOptionActionsRight

You can use the directive *pupaOptionActionsRight for adding any actions in right side of option.

Attribute pupaOptionLink

You can use tag a with the attr pupaOptionLink to use navigation. Also this tag will look and fill as pupa-option .
