Callout

pupa-callout is a static component used to display information for user.

It can be used alone, but it also provides several pre-styled sections:

  • pupa-callout-header - container for callout title
  • pupa-callout-body - container for main content
  • pupa-callout-footer - container for additional links, tags or other components
  • pupa-callout-row - flexible container inside pupa-callout-footer dividing rows
  • pupa-callout-actions - container for action buttons

As well as common action buttons:

  • pupa-callout-collapse-button - button used to shrink callout content
  • pupa-callout-close-button - button used to delete callout from page

All these components are optional.
