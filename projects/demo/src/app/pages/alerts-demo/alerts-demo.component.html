Custom component rendering

Also you can render your own component. Extend it from AlertComponentBase and pass to the open method. Alert data type and return data type will be calculated automatically.

Animations for your own component

You can use default animation or set your own animation. Use AlertComponentBase.animations or write animation like in the example.
