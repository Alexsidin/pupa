Actions

Basic usage

The component renders the passed template based on the length of the action array. When changing the width of the window, an action that does not go crazy in width will be hidden and available in dropdown by trigger This component works on the basis of several building blocks:
1. actions - See API section
2. pupaActionTemplate - Structural directive, used to describe an action template
3. pupaActionMoreTriggerTemplate - Structural directive, used to describe a trigger template for displaying hidden actions
