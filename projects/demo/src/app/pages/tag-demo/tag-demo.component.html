Tag

pupa-tag is a configurable tag component.

There are range of components that are used as content to build and configure the tag itself:

  • pupa-tag-text - a wrapper to truncate long text.
  • pupa-tag-action-button - customized button at left.

pupa-tag-inline - the tag component that makes the content inline.

  • <div pupaTagInline> - you can also use component as tag attribute.
The content inside pupa-tag-inline can be anything, but we recommend using it with text.
