Button Group

Button groups are used to create sets of similar type buttons and can be used as an alternative to radio buttons or as container component to group content.

Basic usage

Full button space

Disabled

Active

Active group button can be set programmatically by the isActive property on a group item. If the property is not specified, the first button in the group becomes active by default.

Group with content

The contents of the group are not cached by default, so when the item becomes inactive, the related content is destroyed. You may use destroyable property on PupaButtonGroupContent to configure this behavior.
