Drag&Drop

Simple usage

Move items in current host and between different hosts (hold down the shift key for moving multiple items)

DndHostComponent

Wrapper for draggable items. Also this component is a drop zone for draggable items of other hosts

DndMoveData

Interface of data, emitted by dndMove event of host element

DndDropData

Interface of data, emitted by dndDrop event of host element

DndItemDirective

Draggable item directive

PupaDndIndicatorComponent

Can be used for highlighting droppable zone of target item
