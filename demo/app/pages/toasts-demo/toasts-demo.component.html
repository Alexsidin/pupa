Template rendering

Change theme

We recommend using light toasts for dark pages and dark toasts for light pages. Also switch theme of toast on page theme changed.

Customization

Custom component rendering

Also you can render your own component. Extend it from ToastComponentBase and pass to the open method. Toast data type and return data type will be calculated automatically.

Animations for your own component

You can use default animation or set your own animation. Use ToastComponentBase.animations or write animation like in the example.
